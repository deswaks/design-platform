Creates and attaches appropriate colliders to a space.

Parameters
spacespace to attach colliders to.

Every collider cube is defined by the controlpoints of the space object. Both the x- and y- position of the collider cube is defined by two indices each that refer to the controlpoints from which to use the {x1, x2, y1, y2 } values from For instance, the location of the collider cube for a rectangular space is defined as follows:

{x1, x2, y1, y2 } = {0, 3, 0, 1} but might also be {1, 2, 3, 2} or {0, 3, 3, 2}.
